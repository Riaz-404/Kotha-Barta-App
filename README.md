# Kotha Barta

![GitHub release (latest by date)](https://img.shields.io/github/v/release/Riaz-404/Kotha-Barta-App?color=f15641)


## Features

- Sign In or Sign Up with email or Google
- Chat with other users
- Real-time message deliver
- Update profile
- Update passwrod
- Delete account
